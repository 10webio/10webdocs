        Installing Plugin Extensions

        Enhance the functionality of your WordPress site even further using powerful extensions. You can integrate Facebook albums into your image galleries, have iCal events imported to event calendars, add advanced calculations to your forms, and much more. 10Web offers these components to help you add supplementary features to the following plugins:

        • WordPress Form Maker,
        • Photo Gallery,
        • Event Calendar WD,
        • Google Maps WD,
        • Ecommerce WD.

        Install them to your website by expanding the extensions section for each of the products mentioned above. You can add multiple extensions by selecting them and pressing the Install button at the bottom of this toolbox.
