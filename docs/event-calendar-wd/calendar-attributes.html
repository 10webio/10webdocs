            Categories, Organizers, Tags And Venues

            Guide to create and modify event categories, organizers, tags and venues.

            Creating Event Categories

            In this section you can add a new event category by entering the category name, slug and description, as well as choose a parent category from the existing ones, add an image and/or specify a category color. There are also search and bulk action options on the left.

            Name. Set a descriptive name for the category.

            Slug. Set a slug using which all of the events using the category will be displayed.

            Parent. Specify a parent category for the event category.

            Description. Provide a short description for the category. This is used for back end only.

            Image Set an image for the category page.

            Category Color. Set a color for the category. The events of the category will be highlighted with the selected color. Keep this in mind when setting the color to make it a fit with the0 calendar theme.

            Creating Event Organizers

            Here you can find the list of all created organizers. You can add new organizers, edit, quick edit, preview or delete them, as well as take bulk actions and/or apply filtering to simplify the search. You can also create new organizers by specifying such details, as:

            Title. Set a name for the event organizer/group of organizers

            Content. Provide details about specific organizers

            Phone. Fill in the phone number of the organizer

            Website. Enter the web-address of the organizer

            Featured image. Set an image from the side-bar to be associated with the specific event organizer.

            Creating Event Tags

            In this section you can add a new event tag by entering the tag name, slug and description. There are also search and bulk action options on the left.

            Name. Set a name for the tag.

            Slug. Set a slug which will be used for displaying the events using the stated tag.

            Description. Provide a short description for the tag, which will help to differentiate the tags.

            Creating Event Venues

            Here you can find the list of all created venues. You can add new venues, edit, quick edit, preview or delete them, as well as take bulk actions and/or apply filtering to simplify the search. Click the ‘Add new’ button to create a new venue.

            You can create new venues by specifying such details, as:

            Title. Set a title for the venue.

            Content. Provide details about the specific venue if necessary.

            Address. Fill in the address of the venue or show the venue location on Google Maps by checking the ‘Show Google Maps’ option. You can drag and drop the Google Maps marker to the location, which will then be displayed with the event. Please note that you need to get a Google Maps key from Google APIs first to be able to configure the maps features and styles in the plugin Settings.

            Phone. Fill in the phone number of the venue.

            Website. Enter the web-address of the venue.

            Featured image. Set an image to be associated with the specific venue.
