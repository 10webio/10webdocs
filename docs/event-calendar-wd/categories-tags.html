                      Event Categories and Tags

                      WordPress Event Calendar plugin allows you to group your events by categories and tags. These features allow you to organize your calendar events and help users navigate through the calendar easily.

                      Letâ€™s begin adding event categories and tags, therewith discussing all available options.

                      Adding Event Categories

                      Navigate to Events > Event Categories page to start adding new categories. This requires just a few simple steps. You need to enter the details listed below:

                      • Name (required),
                      • Slug,
                      • Parent Category,
                      • Description,
                      • Category Icon,
                      • Category Color.

                      Category Color is a handy option which helps you make your event calendar appealing and colorful. By selecting a color with this setting, you also assign the same color for the events belonging to this particular category.

                      Using this setting, Event Calendar WD plugin lets you match the style of the calendar events with your calendar theme.

                      The list of the created categories appears on the right side of this page. Hover over an entry to:

                      • Edit,
                      • Quick Edit,
                      • Delete,
                      • View

                      the categories. You can also search through the created categories, or delete multiple entries using Bulk Actions.

                      Creating Event Tags

                      Click on Events > Event Tags menu item to begin adding new tags to your events. You just need to input the following options:

                      • Name (required),
                      • Slug,
                      • Description.

                      By hovering on a tag from the right side of this page, you can:

                      • Edit,
                      • Quick Edit,
                      • Delete,
                      • View

                      items separately, as well as search through tags and delete multiple entries using Bulk Actions.

                      After creating categories and tags, you can add them to your events. Follow the simple steps described in this section.
