                      Integration Add-ons

                      Description of Event Calendar Integration add-ons.

                      Facebook

                      Do you have an increasing number of events in your Facebook page and want to show them in your website through WordPress Event Calendar. Take advantage of the Facebook Integration Add-on, which will allow you to display Facebook page events within your calendar.

                      Attention: Add-ons are supported by Event Calendar Premium version.

                      For importing/displaying FB events it is required to have admin role permissions of the Facebook page. The imported events will inherit their descriptions, details and location.

                      Displaying event from Facebook page

                      Go to Calendars > Settings > FB Settings. Provide the required details.

                      App ID and App Secret Key: Go to the Facebook Developers Apps page, and sign in with your Facebook username and password. If you are new to Facebook Developers, you will see a “Register a Developer Account” button. Follow the steps to create a free developer account on Facebook.

                      Click the register button. Clicking the “Continue” button implies that you agree to the Facebook Platform Policies. After signing in, click on the “Create New App” button.

                      Specify a name for the application in the “App Name” field. It is recommended to use your site name. Leave the remaining fields blank.

                      Within the “Select how your app integrates with Facebook” section click on “Website” This section will expand to show a “Site URL” field.

                      Enter your site URL in the “Site URL” field (e.g. http://www.example.com ).

                      Click on the “Save Changes” button.

                      You will be redirected to your account page, where both the App ID and App Secret Key will be visible. Facebook events default categories: Set a default category for the displayed Facebook events.

                      Importing events from Facebook page

                      You can import Facebook page events to your calendars. Go to Events > Import FB events, select the options and press “Import” button.

                      The list of the Facebook page events will become visible on the corner of the right side. Here you can select the events you want to import, set category, the calendar (where the events will be added), check the events and press “Save Selected Events” button below.

                      If you import the events from Google Calendar for the second time (or consequently), the events which have already been added to the Calendar will be highlighted. You can choose to re-import them or skip those. Re-import will add an identical event.

                      Google Calendar

                      Have a large list of events in your Google Calendar? Want to display them on your site and customize the design and content of the events through the WordPress event calendar? This add-on makes the entire process of importing events simple and user-friendly. All you need is to provide the Calendar ID and the API key.

                      The add-on has two options: you can display the Google calendar events without storing them in event list. Second option is importing the events to your event calendar. For the second option the events will be added as custom posts(Events) and can be further customized.

                      You can select the events wants to add to the calendar from the available list.

                      The events will maintain their descriptions, details, location and even the settings, e.g. the recurring events will remain recurring and be displayed based on the original recurring rate.

                      Displaying Calendar

                      Note: Make the Google Calendar public. To do so you should go to the Google Calendar page > Calendar setting and set it to be public from “Share this Calendar” section.

                      To display the events from the Google Calendar you should first fill in the details within Event Calendar WD Calendars > Settings > GCal Settings section.

                      Google API key: To fill in this field you should follow the steps below:

                      Go to Google Developers Console and press on “Create new project” button. After the load of the project go to APIs & auth > APIs and look for “Calendar API” and turn the option on. On the left menu click on APIs & auth > Credentials. Within the “Public API access” section click on “Create new key” button and choose “Browser key”. Here provide the list of the domains where you plan to use the calendar on (you can further edit it). The API key will appear. It will take a few second to work, after the activation paste the key to the Google API key field.

                      Google Events default categories: Here you can check the categories of events you want to display from the Google Calendar.

                      Google Events update every: Set the autoupdate time for the Google Calendar events. In cases when a faster update of events are required you can use the Clear Cache option from Settings > General Settings menu.

                      The last step provides the Calendar ID within Calendars > Add/Edit new Calendar section. To obtain the Calendar ID follow the steps here.

                      Importing events from Google Calendar

                      For importing events go to Events > Import GCal, provide Calendar ID, API key choose the calendars to add imported events and press “Import” button.

                      The list of the Google Calendar events will appear on the right side. Here you can select the events you want to import, select their category, as well as the calendar (where the events will be added), check the events and press “Save Selected Events” button below.

                      If you import the events from Google Calendar for the second time (or consequently), the events which have already been imported to the Calendar will be shown with a notice. You can choose to re-import them or skip those events. Re-import will add an identical event.

                      iCal Integration

                      Display the events created using your Apple device on your website effortlessly using the iCal Integration add-on for the WordPress Event Calendar. Start importing iCal event to your website calendar in a simple and practical way.

                      You have created a large number of events using your Apple device and now want to display them on your website? The iCal add-on is designed to manage the process.

                      The add-on allows not only to display the events from the iCloud calendar, but also import those events into the Event Calendar. In this way the events will be turned into custom posts.

                      The number of the displayed events, the date formats can be customized within the Calendar settings.You can select the events you want to add to the calendar from the available list.

                      The imported events will inherit their descriptions, details, location and even the settings, e.g. the recurring events will remain recurring and be displayed based on the original pace.You can also choose the category from which you want to import the events.

                      Note: Make sure that the Apple Calendar to be used is shared publicly. More details can be found here.

                      Displaying Apple Calendar events

                      To display the events from the Apple Calendar you should first fill in the details within Event Calendar WD Calendars > Settings > iCal Settings section.

                      Ical Events default categories: Set the categories that will be used when displaying iCal events.

                      Ical Events update every: Set the autoupdate time for the list of the Apple Calendar events. In cases when a manual update of events is required you can use the Clear Cache option from Settings>General Settings menu.

                      The last step is providing the Calendar ID within Calendars > Add/Edit new Calendar section. To obtain the Calendar URL follow the steps here:

                      Login to iCloud account. Here click on Cloud icon on the top left to get the Menu, choose Calendar. Click on the Cloud icon next to the Calendar you want to use. Here set the Calendar to Share Publicly. The link of the calendar will appear below.

                      Importing events from Apple Calendar

                      As an alternative for displaying the Apple Calendar events within your calendar you can also import the Apple Calendar events. For this please go to Events > Import iCal, select the options and press “Import” button.

                      The Apple Calendar events will be listed on the corner of the right side. Here you can select the events you want to import, set category, the calendar (where the events will be added), check the events and press “Save Selected Events” button below.

                      If you import the events from Apple Calendar for the second time (or consequently), the events which have already been added to the Calendar will be highlighted. You can choose to re-import them or skip those. Re-import will add an identical event.

                      Event Tickets

                      Organize and manage event tickets and invitations easily. Event Tickets Add-on allows your visitors to purchase tickets to your events in a simple and straightforward way directly from your website.

                      Attention: Event Tickets Add-On requires Event Calendar WD Premium version 5.0.34 and higher: if you are using an older version, please update your plugin.

                      General Description

                      Do you use WordPress Event Calendar WD for displaying events? Do you want your site visitors to get tickets and invitations for these events? Then this add-on is the solution combining Event Calendar WD with WooCommerce functionality for selling tickets of your events.

                      The tool is rather easy to use. The integration with WooCommerce plugin is automatic.

                      You can define available dates for ticket sales, price and stock within Event page. Corresponding product will be created based on the filled details in the WooCommerce.

                      The entire payment processed by WooCommerce whereas the tickets are created within Event Calendar WD plugin with all of their details. When the tickets are sold out, visitors will get notified about it with “Sold Out” banner.

                      Note: Make sure to install/activate WooCommerce plugin to your site in order to use ECWD Event Ticket add-on.

                      Settings

                      When the add-on is activated a new set of settings will be added below the event description tab entitled Tickets. Here you should press Add new ticket button. Let’s discuss the option details below:

                      Ticket Title. Provide a title for the ticket. It will be displayed as a product title in WooCommerce product list.

                      Ticket Description. Provide description for the ticket which will be displayed as a product description in WooCommerce.

                      Price. Set the price for the ticket. If you are going to have a free event with invitations you can either leave the price field blank or set it to 0.

                      Starting Date. Choose a date for starting the ticket sales.

                      Ending Date. Choose a date to stop selling the tickets, If you leave it blank the tickets will be available for purchase until the start of the actual event.

                      Stock. Set the total number of tickets for sale. When the available number of tickets get sold out, Sold out banner will appear next to an event.

                      SKU. Set a unique identifying code for each ticket type you’re selling.

                      Payment Options

                      The tickets can be purchased using payment options set by eCommerce plugin. Free version of the plugin supports PayPal only, whereas you can add other payment gateways from the list of WooCommerce extensions.

                      Email Templates

                      After purchasing tickets users will receive two emails - from WordPress WooCommerce plugin and Event Calendar WD. ECWD sends a standard email using default email template which can be found at Event Calendar WD > Calendars > Email Template. You can customize the template making style changes, but make sure to perceive placeholders for normal functionality of the template.

                      Guest Check In

                      The add-on allows using an online check-in for your guests. When purchasing a ticket each user will receive an email containing barcode and security number. You can ask your guests to bring printed emails with them. At an event you can use barcode scanner to scan the tickets or use their security codes for check in.

                      Go to Event Calendar > Events > Event Page > Attendees. This section contains the following details: Order #, Order Status, Purchaser name, Purchaser email, Ticket type, Ticket #, Security Code and Check in. You can check in your guests using two methods- finding them on the list and pressing Check in button next to their name or using the barcode scanning and finding the security code in the list and pressing Check in.
