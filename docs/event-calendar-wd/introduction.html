                      Introducing WordPress Event Calendar

                      Event Calendar plugin is a feature-rich and powerful tool, designed to showcase your calendars and events on your WordPress website. Create unlimited calendars and events, group them by categories, tags, and organizers without having to write a single line of code. The user-friendly settings of the plugin allow anyone, from WordPress beginners to developers, to incorporate event calendars with WordPress.

                      With Event Calendar plugin, you can add unlimited one-time, repeating and multi-day events. Provide your visitors with detailed information by filling in event description, adding links, images and videos, as well as marking up the location on Google Map.

                      The SEO-friendly structure of the plugin lets your events have great search engine visibility. Furthermore, you can place social sharing buttons for each event, allowing website visitors to spread the word on Twitter, Facebook, and Google+.

                      The plugin allows you to fully modify the features and overall look-and-feel of your event calendars. Showcase your events with these 7 stunning views:

                      • Month,
                      • Day,
                      • Week,
                      • List,
                      • Map,
                      • Posterboard,
                      • 4-Day.

                      and personalize their appearance with beautiful customizable themes. The elegant mobile-friendly layout allows you to have your calendars and events appearing flawlessly on any desktop or mobile device.

                      Publish your calendars on any WordPress post or page with an intuitive shortcode toolbox, as well as on sidebars or footer as a widget.

                      Furthermore, you can enhance the features of your event calendars using 14 advanced add-ons. You can display a list of upcoming events or have a countdown for the next event, sell event tickets, and more.
