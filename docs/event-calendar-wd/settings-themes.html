            Calendar Settings And Themes

            Guide to configure Event Calendar plugin Settings.

            Settings

            Here you can configure the global settings of the plugin, including date/time formats, list views, event counts etc., as well as add custom CSS and modify the Google Maps features.

            Themes

            The Event Calendar WD comes with 6 themes. In addition to the original themes, there is a possibility to add a new theme based on the Default theme option. Here you can edit, quick edit, view, duplicate or delete the existing themes, as well as take bulk actions and apply filtering to simplify the search. When editing, you can change the colors, font sizes, borders and other properties of a particular theme.
