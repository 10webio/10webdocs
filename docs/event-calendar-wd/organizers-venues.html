                      Event Organizers and Venues

                      Using WordPress Event Calendar plugin, you have the tools to indicate the organizers of your calendar events, as well as show the location of the venue. You just need to follow a few simple steps to create the organizers and venues, then add them to your events.

                      Adding Event Organizers

                      To begin adding organizers to Event Calendar WD, navigate to Events > Organizers page. The list of all created organizers is provided on this dashboard. You can quickly find entries using the search box, as well as filter items by date.

                      Hover over the organizers to:

                      • Edit,
                      • Quick Edit,
                      • Trash,
                      • View

                      them, or use Bulk Actions drop-down box to edit multiple entries or move them to trash.

                      Click Add New button to create a new organizer. Begin with writing a title and description, giving a few details about the organizing entity. Afterwards, fill in the following options in Organizer Details section:

                      • Phone,
                      • Website,
                      • Featured Image,

                      then click Publish to save the changes. You can add the organizers to your events by following the steps described in this section.

                      Creating Event Venues

                      Click on Events > Venues menu item from WordPress administrative dashboard. You will access the page with all created event venues. From the list of all venues, you can:

                      • Edit,
                      • Quick Edit,
                      • Trash,
                      • View

                      entries separately by hovering on each of them. Use Filter option to view venues by the date they were created. You can also search through the existing entries.

                      Furthermore, you are able to edit multiple venues or move them to trash, using Bulk Actions drop-down box.

                      Press Add new button to create a new venue for your events. Name your venue by entering a title, and write additional information using the content editor box. Afterwards, you just need to fill in the following options and your venue will be ready:

                      • Address,
                      • Phone,
                      • Website,
                      • Show Google Maps,
                      • Featured Image.

                      By marking Show Google Maps checkbox, you can showcase the location of your venue on a beautiful Google Map. And as you fill in Address of your venue, Google Maps marker will be placed on the pointed location. You can also specify Latitude and Longitude of your address.

                      Note, that you need to get Google Maps API key to display a map with your event venue.

                      After you finish configuring venues, click Publish to save the changes. You can add them to your events by following these simple steps.
