            Catalog Global Options

            Instructions to editing WordPress catalog options.

            This section of the WordPress catalog plugin allows making changes within various catalog views, as well as display options, style and colors. Below you will find the list of the Views and corresponding description of the options.

            Global options

            On the left menu select Spider Catalog > Global Options.

            Price. Choose whether the catalog has a price parameter or not.

            Market Price. Choose whether the catalog has a market price parameter or not.

            Currency Symbol. Define the currency of the product price (e.g. ‘$’).

            Currency Symbol Position. Choose whether the currency symbol appears before or after the price number.

            Product Ratings. Choose whether the users can rate the product or not.

            Customer Reviews. Choose whether the users can submit reviews or not.

            Search by Category. Choose whether there is a possibility to filter products by category using the “Choose Category” drop-down list or not.

            Search by Name. Choose whether there is a possibility to search products on the catalog by name or not.
