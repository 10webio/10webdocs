            Creating/Editing Categories for WordPress Catalog Items

            On the left menu select Spider Catalog > Categories > Add a category.

            Name. Add a name for the category.

            Parent Category. You can specify the category as Main Category or as a subcategory (choose the parent category).

            Images. You can add any number of images of the category.

            Description. Add a description for the category.

            Parameters. Here you can define different parameters for the category. You can add as many parameters as you need. After filling in the current parameter title, press “Enter” to proceed to the next parameter.

            Order. Choose the order of the category.

            Published. Choose whether to publish the category or not.

            Creating/Editing WordPress Catalog Items (Products)

            Note: The WordPress Catalog already supports Import CSV and Export CSV possibilities. You can download existing catalog to Microsoft Excel form, manually edit the files and import the updated catalog.

            On the left menu select Spider Catalog > Products > Add Product.

            Name. Choose a name for the product that will appear in the catalog.

            Category. Choose the category/categories for the product.

            Price. Define the actual price of the product.

            Market Price. Indicate the price of the product on the market. It will appear as crossed out. This parameter can also serve to show the initial price of the product or its price before the discount, etc.

            Images. You can add any number of images of the product.

            Description. You can add a description of the product. After filling the short description, you can click on the Read more button below the editor and fill the full description of the product.

            Order. Choose the order of the given product inside its category.

            Show in Parent. If the product belongs to a subcategory, you can set it to appear as part of the parent category.

            Published. Choose whether to publish the product or not.

            Edit Ratings. The product ratings can be edited or removed with this feature. You can find this relevant button on the top left corner.

            Edit Reviews. The product reviews can be removed in this section. You can find this relevant button on the top left corner.
