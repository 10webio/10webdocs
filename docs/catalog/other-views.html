            Catalog List, Thumbnails and Product Page Options

            Instructions to editing WordPress catalog List, Thumbnails and Product Page options.

            List Page

            On the left menu select Spider Catalog > Styles and Colors section. Here you can customize several parameters for different view options.

            In this section you can customize several parameters specifically for the List page view.

            Background Color. Choose the overall background color of the list view.

            Border Style. Pick the border style from the list.

            Border Width. Define in pixels the border width.

            Border Color. Pick the border color of the list view.

            Text Color. Define the color of the body text.

            Category Title Color. Choose the body color for the category title.

            Category Title Background. Select the background color for the category title.

            Category Image Width. Define, in pixels, category image width.

            Category Image Height. Define, in pixels, category image height.

            Category Title Size. Choose the font size for the category title.

            Subcategories List Image Width. Choose the width for the subcategory list images.

            Subcategories List Image Height. Choose the height for the subcategory list images.

            Hyperlink Color. Define the color for the linked texts.

            Search Icon Color. Choose the color for the search icon, appearing in the search box.

            Reset Icon Color. Choose the color for the reset icon.

            Select Category Icon Color. Choose the icon color of the “Select Category” box.

            Rating Star Design. Choose the star design using the preview options.

            Number of Products in the Page. Define the number of products to be visible in the page.

            Categories Background Color 1. Choose the background color for all of the odd products.

            Categories Background Color 2. Choose the background color for all of the even products.

            Categories Header Background Color. Pick the background color for the category header.

            Categories Header Text Color. Choose the color for the category header body.

            Header Text Color. Pick the body color for the main header.

            Header Background Color. Pick the background color for the main header.

            List Background Color 1. Choose the background color for the odd products in the list.

            List Background Color 2. Choose the background color for the even products in the list.

            List Border Color. Choose the border color for the list.

            Price Color. Choose the color for the price tags.

            Market Price Color.C hoose the color for the market price tags.

            Parameter Title Color. Select the title color for all of the parameters.

            Parameter Text Color. Select the body color for all of the parameters.

            Text Size. Define the font size for the main body text.

            Price Size. Define the size for the price tags.

            Market Price Size. Define the size for the market price tags.

            Show Category. Choose whether to show the category or not.

            Show Parameters. Choose whether to show the parameters or no.

            Show Description. Choose whether to show the descriptions for the products or no.

            Thumbnails Page

            On the left menu select Spider Catalog > Styles and Colors section. Here you can customize several parameters for different view options.

            In this section you can customize several parameters specifically for the Thumbnails page view.

            Background Color. Specify the background color of the overall page, where the catalog is used.

            Border Style. Choose the border style for the thumbnail view option from the provided 8 options.

            Border Width. Specify, in pixels, the border width.

            Border Color. Choose the color to be applied to the borders.

            Text Color. Define the font color for the main body.

            Category Title Text Color. Select the color shade for the category titles.

            Category Title Background Color. Select the background color shade for the category titles.

            Category Image Width. Specify the category image width.

            Category ImageHeight. Specify the category image height.

            Category Title Size. Define the title size for the category.

            Subcategories List Image Width. Define the pixel width for the subcategory list images.

            Subcategories List Image Height. Define the pixel height for the subcategory list images.

            Hyperlink Color. Specify the color for the hyperlinks.

            Search Icon Color. Define the color for the search icon.

            Reset Icon Color. Select the color for the search icon.

            Select Category Icon Color. Define the color for the Select category feature.

            Rating Star Design. Select the star design to be used for the rating from the provided options.

            Number of Products in the Row. Limit the number of visible products per page.

            Number of Rows in the Page. Limit the number of product rows per page.

            Categories Background Color 1. Pick the background color for the odd products.

            Categories Background Color 2. Pick the background color for the even products.

            Categories Header Background Color. Select the header background color.

            Categories Header Text Color. Define the font color for the category headers.

            Title Size. Define the main text font size.

            Title Color. Choose the main title color.

            Price Size. Specify the font size of the price.

            Price Text Color. Choose the font color for the price.

            Image Width. Define the width of the main images.

            Image Height. Define the height of the main images.

            Text Size. Choose the font size for the main body text.

            Text Color. Specify the font color for the main body text.

            Cell Thumbnail Width. Define the width for the thumbnail cells.

            Cell Thumbnail Height. Define the height for the thumbnail cells.

            Cell Background Color. Select the background color for the overall cells.

            Product page

            On the left menu select Spider Catalog > Styles and Colors section. Here you can customize several parameters for different view options.

            In this section you can customize several parameters specifically for the Product page view.

            Background Color. Apply a background color to the page from the different colors and shades.

            Border Style. Use the drop-down list to select the border style.

            Border Width. Specify the border width in pixels for the single products.

            Border Color. Select the border color.

            Text Color. Choose the color for the main text.

            Text Size. Determine the font size for the main text.

            Parameters Background Color 1. Select the background color for the odd product parameters.

            Parameters Background Color 2. Select the background color for the even product parameters.

            Color of Parameter Values. Choose the color of the detailed parameters.

            Price Color. Define the color for the price line.

            Market Price Size. Specify the font size of the market price.

            Market Price Color. Define the color for the market price line.

            Price Size. Specify the font size of the price.

            Price Background color. Select the background color for the price.

            Title Color. Choose the color of the product title.

            Big Title Size. Specify the font size for the first line of the title.

            Title Size. Specify the main text font size.

            Title Background Color. Choose the background color for the product title.

            Image Width. Define the width for the single products.

            Image Height. Define the height for the single products.

            Rating Star Design. Choose the star design for the rating system.

            Background Color of Review Section. Select the background color for the review section.

            Color of Review Texts. Choose the color shade for the review provided by the users.

            Background Color of “Review Author” section. Choose the background color to be used with Review Author box.

            Background Color of Review Text. Choose the background color to be used with the review box.

            Number of Reviews per Page. Determine the number of visible reviews per page.

            Background Color of “Add your review here” and “Send” Buttons. Select the background color for the “Add your review here” and “Send” buttons.

            Text Color of “Add your review here” and “Send” Buttons. Choose the font color for the “Add your review here” and “Send” button labels.
