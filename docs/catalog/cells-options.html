            Catalog Cells Page Options

            Instructions to edit WordPress catalog Cells Page options.

            Cells1 Page Options

            On the left menu select Spider Catalog > Styles and Colors section. Here you can customize several parameters for different view options.

            In this section you can customize several parameters specifically for the Cells1 page view.

            Background Color. Select the overall background color for the cells1 option.

            Border Style. Choose the border style of the item cell.

            Border Width. Define the cell border width of the item.

            Border Color. Select the border color.

            Text Color. Select the color for the description texts, parameter titles, and other field titles.

            Category Title Text Color. Choose the font color for the category titles.

            Category Title background color. Choose the background color for the category titles.

            Category Image Width. Define the category image width.

            Category Image Height. Define the category image height.

            Category Title Size. Define the title font size for the used categories.

            Subcategories List Image Width. Define the image width for the listed subcategories.

            Subcategories List Image Height. Define the image height for the listed subcategories.

            Image Width. Define the width of the product image.

            Image Height. Define the height of the product image.

            Hyperlink Color. Choose the color of hyperlinks appearing in the item cell.

            Search Icon Color. Choose the icon color of the search button.

            Reset Icon Color. Choose the icon color of the reset button.

            Select Category Icon Color. Choose the color for the Select Category icon.

            Number of Products in the Row. Choose the number of the product that will be displayed in a single row.

            Number of Rows in the Page. Choose the number of the rows that will be displayed in a single page.

            Rating Star Design. Select the star rating design using the preview possibility.

            Parameters Background Color 1. Choose the background color for the odd product parameters.

            Parameters Background Color 2. Choose the background color for the even product parameters.

            Price Color. Choose the color for the price.

            Market Price Color. Choose the color for the market price.

            Title Color. Choose the color for the main title.

            Title Background color. Choose the background color for the main title.

            Categories Background Color 1. Select the background color for every odd product parameter.

            Categories Background Color 2. Select the background color for the even product parameters.

            Categories Header Background Color. Choose the background color of the category header.

            Categories Header Text Color. Choose the font color for the category header.

            Big Title Size. Define the font size of the first line of the product title.

            Small Title Size. Define the font size of the second line of the product title. This feature only affects when the title is long and takes more than one line.

            Price Background Color. Select the price background color.

            Cell Background Color. Choose the background color for the overall cell view.

            More Text Color. Select the text color for the More button.

            More Background Color. Select the background color for the More button.

            Parameter Text Color. Define the body text color for the parameter titles.

            Parameter Title Font Size. Define the font size for the parameter titles.

            More Font Size. Select the font size for the More button.

            Product Cell Width. Define the width for the product cell.

            Product Cell Height. Define the height for the product cell.

            Price Size. Define the font size for the price.

            Market Price Size. Define the font size for the market price.

            Show Category. Choose whether to display product category in product cell or not.

            Show Parameters. Choose whether to display product parameters in product cell or not.

            Cells2 Page Options

            On the left menu select Spider Catalog > Styles and Colors section. Here you can customize several parameters for different view options.

            In this section you can customize several parameters specifically for the Cells2 page view.

            Background Color. Select the background color of the page using the cells2 view option.

            Border Style. You can choose a specific border style for this view option or leave it to default style.

            Border Width. Define the border width of the separate cells.

            Border Color. Select the color shade to be used for the border.

            Text Color. Select the color for the main body.

            Category Title Text Color. Choose the color to be applied to the category titles.

            Category Title Background Color. Choose the background color to be applied to the category titles.

            Category Image Width. Specify the category image width.

            Category Image Height. Specify the category image height.

            Category Title Size. Specify the title font size for the category.

            Subcategories List Image width. Define the width for the subcategory list images.

            Subcategories List ImageHeight. Define the height for the subcategory list images.

            Hyperlink Color. Define the color for the hyperlinks.

            Search Icon Color. Choose the icon color for the search.

            Reset Icon Color. Choose the icon color for the reset.

            Select Category Icon Color. Choose the icon color for the Select Category feature.

            Rating Star Design. Choose the star design to be used for the rating.

            Number of Products in the Row. Here you can provide the number of products to be visible in the rows.

            Number of Rows in the Page. Here you can provide the number of rows to be visible in the page.

            Categories Background Color 1. Choose the background color for the odd products.

            Categories Background Color 2. Choose the background color for the even products.

            Categories Header Background Color. Pick the background color for the category header.

            Categories Header Text Color. Pick the font color for the category header.

            Big Title Size. Define the font size for the first line of the title.

            Title Size. Define the main text font size.

            Title Color. Select the main title color.

            Price Size. Specify the font size of the price detail.

            Price Color. Specify the color for the price color.

            Market Price Size. Define the font size of the market price detail.

            Market Price Color. Define the color of the market price detail.

            Image Width. Specify the width of the main images.

            Image Height. Specify the height of the main images.

            Text Size. Determine the font size for the main texts.

            New Cell Width. Define the width for the cells.

            New Cell Height. Define the height for the cells.

            Cell Background Color. Choose the color option for the cell background.

            Text Color. Determine the color to be used for the main texts of the view.

            Text Background Color 1. Choose the background color for list of the odd products.

            Text Background Color 2. Choose the background color for the list of even products.

            More Size. Define the size of the More button.

            More Text Color. Define the font color of the More button label.

            More Background Color. Select the background color for the More button.

            Border Color. Select the color option for the borders.

            Text Color. Choose the color for the main body text.

            Category Title Text Color. Select the color for the category title.

            Category Title Background Color. Select the background color for the category title.

            Subcategories List Image Height. Define the height of the subcategory list images.

            Hyperlink Color. Choose the color for the hyperlinks.

            Search Icon Color. Select the color for the search icon.

            Reset Icon Color. Select the color for the reset icon.

            Select Category Icon Color. Select the color for the Select Category icon.

            Rating Star Design. Choose the star design to be used for the rating.

            Number of Products in the Page. Determine the number of products you want to make visible in a single page.

            Categories Background Color 1. Select the background color for the odd products.

            Categories Background Color 2. Select the background color for the odd products.

            Categories Header Background Color. Select the background color for the products categories.

            Categories Header Text Color. Define the font color for the category headers.

            Cell Background Color. Define the background color for the overall cell.

            Title Size. Define the main text font size.

            Title Color. Choose the main title color.

            Price Size. Specify the font size of the price tag.

            Price Color. Specify the color for the price detail.

            Text 1 Size. Specify the font size for the main body text for the odd products.

            Text 2 Size. Specify the font size for the main body text for the even products.

            Parameter Text Color 1. Choose the font color of the parameters for the odd products.

            Parameter Text Color 2. Choose the font color of the parameters for the even products.

            Parameter Background Color 1. Select the background color for the parameters of the odd products.

            Reset Icon Color. Choose the color shade for the search icon.

            Select Category Icon Color. Choose the color shade for the Select Category line.

            Rating Star Design. Select the star design you want to use for the rating system.

            Number of Products in the Row. Define the number of products per row.

            Number of Rows in the Page. Define the number of rows in the page.

            Categories Header Background Color. Choose the background color for the category header.

            Categories Header Text Color. Select the font color for the category header.

            Title Size. Specify the main text font size.

            Title Color. Select the main title color.

            Price Size. Define the font size of the price.

            Price Text Color. Specify the color of the price detail.

            Image Width. Define the width of the main images.

            Image Height. Define the height of the main images.

            Cell Background Color. Select the background color for the large cell.
