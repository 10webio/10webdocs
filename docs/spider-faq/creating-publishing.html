      Creating and Publishing FAQs

      Instructions to create and publish FAQs, categories and questions with Spider FAQ plugin.

      Creating categories

      From the navigation bar on the left select FAQ > Categories.

      Click on the Add a Category button on the upper right hand of the screen.

      Title. Add a title for the category.

      Description. Add a description for the category.

      Show Title. Choose whether to show the title or not.

      Show Description. Choose whether to show the description or not.

      Published. Choose whether to publish the category or not.

      Creating FAQs

      From the navigation bar on the left select FAQ > Questions.

      Click on the Add a Question button on the upper right hand of the screen.

      Question. Add a question.

      Category. Choose a category for the question from the list.

      Answer. Add the answer to the question.

      Created by. Click on Select User button to add the user, who contributed the question. By default it shows the username of the logged in user. You can also leave it blank.

      Calendar. Choose the date when the question was contributed.

      Like. You can edit the number of likes here.

      Unlike. You can edit the number of unlikes here.

      Hits. You can edit the number of hits here.

      Order. Select the order of the question.

      Published. Choose whether to publish the question or not.

      Creating an FAQ section

      From the navigation bar on the left select FAQ > FAQs.

      On the top right hand of the screen click on the Add an FAQ button.

      Title. Specify a title for the FAQ section.

      Theme. Choose a theme for the FAQ from the list.

      Use Standard Category. If this option is enabled, the existing post categories will also be included in the list of available categories.

      Select category. Choose a category from the list.

      Show Search Form. Choose whether to show the search form or not.

      Expand All Answers After The Page Is Loaded. Choose whether to expand all answers after the page is loaded.

      Category numbering. Choose whether to have a numbering for the categories or not.

      Like. Choose whether to display like/unlike buttons with FAQ section or not.

      Hits. Choose whether to display the number of hits or not.

      Date. Choose whether to display the date, when the question was submitted or not.

      User. Choose whether to display the name of the user, who contributed the question or not.

      Publishing FAQs in a page or a post

      Open the page/post where you want to insert the FAQ.

      Click on the Insert Spider FAQ button (the button with a question mark on it).

      Select the FAQ you want to insert and click on the Insert button.

      Alternatively, you can manually insert the FAQ shortcode into the post/page. Simply copy [Spider_FAQ id="N"] into the page/post. “N” is the ID of the FAQ and can be found in the FAQs section of the Spider WordPress FAQ.
