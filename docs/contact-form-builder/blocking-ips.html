            Blocking IPs

            Instructions to block spammer IP Addresses with Contact Form Builder plugin

            • Go to Plugins > FormBuilder > Blocked IPs.
            • Click on the New button.
            • IP Provide the IP address you want to block.

            Note: Another option of blocking IPs can be done from the submissions section. You should go to Form Builder > Submissions. Check the box in front of spam submissions and press Block IPs button at the top right corner.
