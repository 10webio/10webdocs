            Editing Contact Form Builder

            Instructions to edit Contact Form Builder forms.

            Go to Contact Form Builder > Manager.

            Select the Contact Form you want to make the relevant changes. Click on the title to open the specified form.

            Navigation Buttons. 4 navigation arrows are displayed next to each field. You can use those arrows to move the fields up, down, left or right.

            Note: for larger fields to get the idea of the form display in the front-end you should press Collapse menu on the lowest left corner.

            Edit button. Using edit button, you will be able to apply specific changes in the field, including titles, labels, dimensions and etc.

            Checkbox. Using the checkboxes next to the fields you can enable or disable the provided field.
