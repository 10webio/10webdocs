              Creating Map

              Instructions and description on creating a Google Map.

              Maps are listed in Google Maps WD > Maps section, where you can add/edit maps. Several maps are created automatically an.

              After titling the map you can view on three tabs below.

              Map: here you can add attributes on map (markers, polygons, circles, rectangles, polygons and polylines) and modify them. Note that Center address and Zoom Level of this map will the same is in frontend. Settings: Here you can configure settings of the map: general options, map controls, layers, direction listings, store locator and marker listing. Theme/Preview: Here you can preview the map using one of the predefined themes.
