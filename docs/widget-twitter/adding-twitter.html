      Adding Twitter social plugins

      Adding WordPress Twitter plugin to the website.

      This section allows adding Twitter buttons, timelines and etc. Below you will find the list of possible buttons and timelines with detailed description.

      From the navigation bar on the top select Plugins > Widget Twitter. On the upper right hand of the screen click on the Add New button.

      Adding a Tweet Button

      Title. Specify a title for the plugin for identifying the plugin in the back-end.

      Published. Choose whether to publish the plugin or not.

      Via User. Specify the screen name of the user to which the tweet will be attributed to.

      Tweet Text. Type in the tweet text. If you leave it blank, it will automatically use the page title.

      Type of URL. Choose the URL of the page to share. You can either choose to leave the current URL or provide another URL.

      URL to Tweet. Provide the URL to share with the tweet.

      URL to which your shared URL resolves. Specify the URL of the page which will be associated with the tweet. It will not be visible as a link but will be counted with the help of count box and will return all the tweets when the link is searched. E.g. you can provide a link to the homepage of your website to be visible with the count box.

      Hashtag. Specify hashtags for your tweet. You can use multiple hashtags separating them with commas.

      Button Size. Choose the button size from the provided default options.

      Count Box Position. Choose whether to use a count box next to the Tweet button as well as its positioning between horizontal and vertical options. Count Box states the amount of the mentions for the resolved URL.

      Opt Out. Check this option to avoid Twitter from content tailoring and making suggestions for you and your followers based on the data.

      Language Preference.

      • Current. Leave to the language of the website.
      • Custom. Select the plugin language.

      All Posts. Check the option to add the plugin to all the posts on the website.

      Posts. Choose the posts to which you want to add the plugin.

      Vertical Position. Choose whether to place the plugin at the top or at the bottom of the article.

      All Pages. Adds the plugin to all the pages on the website.

      Pages. Choose pages to which you want to add the plugin.

      Vertical Position. Choose whether to place the plugin at the top or at the bottom of the article.

      Style. Here you can customize certain style options. Simply change the values of the parameters that are listed in the box.

      Adding a Follow Button

      Title. Specify a title for the plugin for identifying the plugin in the back-end.

      Published. Choose whether to publish the plugin or not.

      User to follow. Define the user screen name to follow, when pressing the button. By default the screen name will be visible when hovering.

      Button Size. Choose the button size from the provided default options.

      Followers count display. Choose whether to display the number of the followers in the box or not.

      Width. Define the width of the Follow button. The height of the button is set by default.

      Alignment. Choose the alignment of the button.

      Show Screen Name. Check the box if you want the screen name to be visible on the button.

      Opt Out. Check this option to avoid Twitter from content tailoring and making suggestions for you and your followers based on the data.

      Adding a Timeline Box

      Choose a timeline source. Choose the timeline source from the provided 4 options.

      User Timeline. You can choose to create a timeline based on public data of a specific user. The header of the timeline will display a follow button to follow the user, while the footer will allow sending a message to the user without leaving the page.

      Favorites. You can create a timeline based on the favorite tweets of a specific user.

      List. You can create a timeline based on a list of specific users. The header of the timeline will include the list name, its description, as well as the screen name of the user, who created the list.

      Search. You can create a timeline based on a certain hashtag. The timeline will display the search results based on it. The hashtag will be displayed on the header of the timeline, while the footer will allow contributing to the list making a tweet with the hashtag. The list of the search results can be refined to the safe mode, avoiding the display of certain results.

      Title. Specify a title for the plugin for identifying the plugin in the back-end.

      Published. Choose whether to publish the plugin or not.

      Widget ID. To create a widget ID, first login to your Twitter account, then you should go to widget settings page, click on edit button for the specified widget and copy the number code, which appears in the URL on the top.

      Width. Define the width of the button in pixels.

      Height. Define the height of the button in pixels.

      Tweet Limit. Specify the number of tweets to be displayed with the timeline. This option will remove the sidebar and cannot be used with defined height.

      Opt Out. Check this option to avoid Twitter from content tailoring and making suggestions for you and your followers based on the data.

      Theme. Choose between dark and light default theme options.

      Link Color. You can customize the color of the links that appear in the timeline to match the overall theme used on the page.

      Border Color. You can customize the border color of the displayed timeline.

      Chrome. Use a space-separated set of the following options:

      • No header. Hides the timeline header.
      • No footer. Hides the timeline footer and Tweet box, if included.
      • No borders: Removes all borders within the widget (between Tweets, around the widget.)
      • No scrollbar: Crops and hides the main timeline scrollbar, if visible.
      • Transparent: Removes the background color.

      ARIA politeness. By default the ARIA politeness is set to polite, but you can make it assertive.

      Show Replies. The option is available with User timeline and allows to add the replies to the tweets in the timeline.
