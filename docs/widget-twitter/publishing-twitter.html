      Publishing Twitter social plugins

      Publishing Twitter social plugins on the website

      Publishing Widget Twitter as a Widget

      Go to Appearance > Widgets. Here select Widget Twitter, drag and drop it into the custom location, fill in the options and press Save button.

      Details

      Title. Provide the name of the widget.

      Select Plugin. Select the WordPress Twitter plugin you want to insert.

      Publishing **Widget Twitter **plugins in a page or a post

      Open the page/post where you want to insert the WordPress Twitter plugin.

      Click on Insert Widget Twitter button (Twitter Icon).

      Alternatively, you can manually insert the Twitter plugin shortcode into the post/page. Simply copy [Widget_Twitter id="N"] into the page/post. “N” is the ID of the Twitter plugin and can be found in the Widget Twitter section of the Widget Twitter.
