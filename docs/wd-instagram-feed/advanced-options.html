      Styling with Themes
        Advanced Customizing Options

          Instagram Feed WD comes with detailed themes section, but if you want to make specific style changes you can also use

          • Custom CSS field located in Instagram Feed > Settings.
          • Custom Javascript within the same location.

          A Custom Javascript event is available for scripts to run after all feed images are loaded. Suppose you have feed with thumbnails view and besides displaying your feed you want to open up captions for some photos. You can achieve this by assigning our custom wdi_feed_loaded event to feed container and set callback function for that event. Here is quick example how to open up captions for images with 0,1,2,3 indexes.

          jQuery('#wdi_feed_0').on('wdi_feed_loaded',function(){
    jQuery('.wdi_photo_title').eq(0).trigger('click');
    jQuery('.wdi_photo_title').eq(1).trigger('click');
    jQuery('.wdi_photo_title').eq(2).trigger('click');
    jQuery('.wdi_photo_title').eq(3).trigger('click');
  });
