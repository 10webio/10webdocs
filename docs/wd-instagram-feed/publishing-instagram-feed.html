          Publishing Instagram feed

          Instructions to publish WordPress Instagram feed.

          You can publish Instagram Feed in a post/page, as a widget or using php function.

          Publishing in a Page/Post

          To use the created feeds within a page or post you should use shortcode. Open the page/post from the back end. Press on the Add Instagram Feed button within the editor. Then select the feed and press Insert button. An alternative will be copying the shortcode from Instagram Feed WD> Feeds toolbar and pasting it within post/page.

          Publishing as a Widget

          Go to Appearance > Widgets toolbar.

          Find Instagram Feed WD widget and assign it to a custom widget position. Fill in the details below:

          Title. Provide a title for your Instagram Feed.

          Feed. Select the feed which will be displayed as a widget.

          Number of Columns. Set the number of image columns (for thumbnail and masonry view).

          Number of images to display. Set a constant number of images which will be displayed with widget.

          Show Likes and Comments. Check the box to display likes and comments below view thumbnails.

          Publishing by PHP function

          It is also possible to insert the created feeds into the theme by calling it with PHP function. This feature is commonly used for adding the code to header.php for displaying feeds within header or footer position of the website. You can find the function located in Instagram Feed > Feeds section under PHP function column.
