        FAQ Themes

        Instructions to edit FAQ Themes in FAQ WD plugin.

        In this section of WordPress FAQ plugin you can make changes to the default themes add create new themes.

        General Settings

        In this subsection you can change the general, search box, expand/collapse and read more parameters. You can change the general width, text colors, font sizes and set margin.

        Categories

        In this subsection you can change category-related settings. You can change the text color, background color font size, as well as set borders and define its parameters. The same applies to the category description, which in addition to the first list can also have margin and padding values. The selected category can have a different color as well as border.

        Question Settings

        In this subsection you can change question-related settings. You can change the space between the question, text color and hover color, background color, font size, as well as set borders and define its parameters. If the questions are numbered you can set the font size and color for the numbers. The default navigation arrows can be replaced with custom buttons when uploading the images.

        Answer Settings

        In this subsection you can change answer-related settings, as well as parameters of comments, views, date display and votes (likes). You can change the text color and hover color, background color, font size. In case of the answer you can set margin and borders and define their parameters.
