            Photo Gallery Options

            Editing Photo Gallery Options

            Global Options

            Images directory. Provide an image directory location which will host the uploaded images.

            Image Dimensions. Provide the maximum acceptable dimensions for the image, which can be uploaded to the gallery.

            Right-click protection. Choose whether to enable the right-click protection for the images or not.

            Show Search box. Choose whether to display a search box for the image search in galleries or not.

            Search box width. Set the width for the search box.

            Show “Order by” dropdown list. Choose whether to display a dropdown list allowing the user to change the ordering option of the displayed images or not.

            Show tag box. Choose whether to display a search box for tags used with galleries or not next to the gallery.. When searching for the tag the images using that tag will be displayed.

            Preload images. Choose whether to have a preload for specific number of images first or have them all loaded and only then displayed in the slideshow.

            Count of images. Specify the number of images which will be loaded before opening the slideshow. The rest of the images will be loaded during the processing of the slideshow.

            Import from Media Library. Choose whether to allow adding images from Media Library of WordPress or not.

            Enable href attribute. Enable the option for including images in sitemaps and searches.

            Meta auto-fill. Enable the option for generation of automatic metadata for the images based on camera-included data.

            Watermark

            Watermark. You can use this option of Photo Gallery to add different types of watermarks on your images.

            Watermark Type. Choose what kind of advertisement you want to use from provided 2 options (text, image) or not to use watermarks in your galleries and albums.

            Text

            • Watermark text. Provide the text which will be displayed over the gallery/album images.
            • Watermark font size. Specify the font size of the watermark in percentage for the responsive look.
            • Watermark font style. Specify the font family for the watermark text. There will be two default font styles to select from, but you will be able to further add more styles by inserting the fonts into the “fonts” folder in the Plugin folder.
            • Watermark color. Select the watermark text font color.
            • Watermark opacity. Define the level of opacity for the watermark, providing its percentage.
            • Watermark position. Choose the positioning of the watermark from the available 9 options.

            Image

            • Watermark URL. Provide the absolute URL of the image which will be used for the watermark.
            • Watermark image size. Define the size (in percentage for the responsive look) of the image used for the watermark.
            • Watermark position. Choose the positioning of the watermark from the available 9 options.

            Advertisement

            You can use Advertisement option of Photo Gallery to add advertisement to your website both for videos and images.

            Advertisement Type. Choose what kind of advertisement you want to use from the provided 2 options (text, image) or not to use ads in your galleries and albums.

            Text

            • Advertisement text. Provide the text which will be displayed over the gallery/album images.
            • Advertisement link. Provide the link, which will open in a new window, when the user clicks on advertisement text.
            • Advertisement font size. Specify the font size of the advertisement text.
            • Advertisement font style. Specify the font family for the text used with the advertisement.
            • Advertisement color. Select the font color which will be used with the advertisements.
            • Advertisement opacity. Define the level of opacity for the advertisement, providing its percentage.
            • Advertisement position. Choose the positioning of the advertisement from the available 9 options.

            Image

            • Advertisement URL. Provide the absolute URL of the image which will be used for the advertisement.
            • Advertisement link. Provide the link, which will open in a new window, when the user clicks on the advertisement image.
            • Advertisement dimensions. Define the dimensions of the image used for the advertisement.
            • Advertisement opacity. Define the level of opacity for the advertisement, providing its percentage.
            • Advertisement position. Choose the positioning of the advertisement from the available 9 options.

            Lightbox

            Full width lightbox. Enable full width features for the lightbox.

            Ligthbox dimensions. Define the width and height of the lightbox.

            Lightbox effect. Select the effect which will be applied to the slideshow of images in the lightbox.

            Lightbox autoplay. Choose whether to autoplay the lightbox or not.

            Time Interval. Provide the time interval between the display of slideshow images in seconds.

            Enable filmstrip. Choose whether to include the filmstrip of images in the lightbox or not.

            Filmstrip height. Define the height of the filmstrip in pixels.

            Display hit counter. Choose whether to display hit counter for the image ratings/reach.

            Enable control buttons. Choose whether to display control buttons in the lightbox or not.

            Enable fullscreen. Choose whether to have a possibility of entering Fullscreen from the lightbox or not.

            Enable info. Choose whether to have possibility of displaying image information (Title and description) in this view or not.

            Display info by default. Choose whether to display image title by default when opening the lightbox or not. The image can be hidden hitting info button.

            Enable rating. Choose whether to display rating possibility for the images or not.

            Enable comments. Choose whether to display commenting possibility with the images or not.

            Enable comments moderation. Choose whether to allow removing/moderating comments by admin or not.

            Enable Email for comments. Choose whether to have required email field for the users planning to add comments or not.

            Enable Captcha for comments. Choose whether to have Captcha field for the possibility of adding comments as a human verification or not.

            Enable original image display button. Choose whether to allow displaying original images upon clicking the button or not.

            Enable download button. Choose whether to allow downloading the images or not.

            Enable Facebook button. Choose whether to display Facebook share button with the images or not.

            Enable Twitter button. Choose whether to display twit button with the images or not.

            Enable Google+ button. Choose whether to display Google+ share button with the images or not.

            Enable Pinterest button. Choose whether to display Pinterest pin button with the images or not.

            Enable Tumblr button. Choose whether to display Tumblr share button with the images or not.

            Show Image Count. Choose whether to display the number of the images in the lightbox or not.

            Enable Loop. Choose whether to move to the first image after the image roll is finished or not.

            Enable AddThis. Choose whether to display AddThis bookmarking button with the image or not.

            AddThis Profile ID. Whenever adding AddThis bookmarking option you should provide your AddThis profile ID.

            Album Options

            Number of album columns. Provide the maximum number of columns to be displayed within a single album.

            Albums per page. Provide the maximum number of albums available in a single page.

            Enable pagination. Choose whether to provide pagination for the albums or not.

            Show album title. Choose whether to always display the album title, have it displayed only when hovered or never display it.

            Enable extended album description. Choose whether to have extended album view option or not.

            Album thumbnail dimensions. Define the width and height of the album thumbnails.

            Extended album height. Define the height of the album in extended album view.

            Slideshow

            Slideshow effect. Select the slideshow effect to be applied to the galleries.

            Interval. Provide the time interval between the slideshow images in seconds.

            Slideshow dimensions. Define the width and height of the slideshow.

            Enable autoplay. Choose either to have the slideshow auto played or allow the users to change the images with a click.

            Enable shuffle. Choose whether to have the images randomized in the slideshow or not.

            Enable control buttons. Choose whether to display control buttons with image slideshow views or not.

            Enable slideshow filmstrip. Choose whether to include the filmstrip of images in the slideshow or not.

            Slideshow filmstrip height. Define the height for the filmstrip for the slideshow.

            Enable image title. Choose whether to display the titles for the slideshow images or not.

            Title position. Select the positioning for the image title from the provided options.

            Enable image description. Choose whether to have an image description or not.

            Description position. Select the positioning for the image description from the provided options.

            Enable slideshow music. Choose whether to have an audio track for the slideshow view or not.

            Music URL. Provide the URL of the track which will be played alongside the slideshow.

            Thumbnail Options

            Masonry. Choose whether to have horizontal or vertical masonry view for the masonry view thumbnails.

            Mosaic. Choose whether to have horizontal or vertical arrangement of the thumbnails for the mosaic view.

            Resizable mosaic. Choose whether to resize the thumbnails when the gallery page gets loaded on a smaller device or not.

            Total width of mosaic. Define the area (in percentage) that can be taken by the mosaic gallery proportionate to the content container.

            Number of image columns. Provide the maximum number of image columns to be displayed with Thumbnails view.

            Images per page. Provide the maximum number of images to be displayed in a single page.

            Generated thumbnail dimensions. Provide the dimensions of the thumbnail image which will be generated from the original image.

            Frontend thumbnail dimensions. Provide the dimensions of the image, which will be displayed in the front end as a thumbnail. This should be smaller than the generated thumbnail to keep the quality of the image.

            Show image title. Choose whether to always display the image title, have it displayed only when hovered or never display it.

            Show gallery name. Choose whether to display the Gallery titles over the thumbnail images.

            Show description in Vertical Masonry view. Choose whether to display the description of the images in Vertical Masonry view or not.

            Enable image pagination. Choose whether to have image pagination or not.

            Thumb click action. Choose the action when hitting the image thumbnail. Whenever using redirect URL, you should return to the gallery and list the URL to be redirected for every individual image in the Alt title section in the second textbox.

            Open in a new window. Choose whether to open the URL in a new window or not.

            Play icon over the video thumbnail. Choose whether to display Play triangle icon over the videos or not.

            Image Options

            Enable image title for Image Browser view. Choose whether to display image titles for the Image Browser view or not.

            Enable image description for Image Browser view. Choose whether to have image descriptions for the Image Browser view or not.

            Image width. Define the image width for the Image Browser view option.

            Enable image title for Blog Style view. Choose whether to have image titles in Blog Style view or not.

            Image width for Blog Style view. Define the image width for the Blog Style view.

            Images per page in Blog Style view. Specify the number of images to be displayed in a single page for the Blog Style view.

            Enable Pagination for Blog Style view. Choose whether to have pagination for the Blog Style view or not.

            Social Options

            Gallery Autoupdate interval. Set time interval for updating all galleries set to autoupdate.

            Instagram CLIENT ID. You should follow the steps below to get the Client ID:

            1. Login to Instagram.
            2. Go to instagram.com/developer which is Instagram developer’s page.
            3. Press ”Register your application” button.
            4. If you have already used the Client registering you will see the Client ID, otherwise you should press ”Register new client” button on the top right.
            5. Provide the application name, description and website (for the last two URL lines ).
            6. Press Register button.
            7. Click ”Manage clients” in the top menu and you will now see you new client in the list and a client id, client secret and more. Copy the client_id and past that to Client ID section in the Options menu.

            Autoupdate interval. Set time interval for updating all galleries set to autoupdate.

            Carousel Options

            Time interval. Set the timing which will be used for image carousel roll.

            Max. number of images. Limit the amount of image visible for the carousel view.

            Carousel image ratio. Define the proportions between the main image and images preceding and following the main image in the carousel roll.

            Image dimensions. Set the dimensions for the centered image of the carousel roll.

            Fixed width. Define the maximum width of the carousel display.

            Enable image title. Choose whether to display the image title over the main image or not.

            Enable autoplay. Choose whether to enable the autoplay of the carousel roll or not.

            Container fit. If you activate the option the distance between the images will be identical so that the images fill the container, otherwise the distance between the images will depend on the ratio.

            Next/Previous buttons. Choose whether to include navigation buttons for the roll or not.

            Play/Pause button. Choose whether to display play/pause buttons over the carousel roll or not.
