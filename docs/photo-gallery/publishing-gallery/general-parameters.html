            General Parameters

            Publishing Photo Gallery General Parameters.

            Thumbnails

            Theme. Select the theme to be applied to the view.

            Gallery. Select the gallery.

            Sort by. Choose the option to use for image sequence.

            Order. Set the ordering of the gallery images.

            Show search box. Choose whether to display search box or not.

            Show “Order by” dropdown list. Choose whether to display a dropdown list allowing the user to change the ordering option of the displayed images or not.

            Max. number of image columns. Provide the number of image columns in a page.

            Images per page. Provide the maximum number of images per page.

            Image title. Choose whether to display the image title (always/on hover) or not to display it at all.

            Enable pagination. Choose whether to have pagination for the view or not. You can also use “Load More” option which will load more images based on the amount specified in images per page option or choose image loading upon scrolling.

            Image thumbnail dimensions. Define the thumbnail width and height in pixels.

            Mosaic

            Theme. Select the theme for the view.

            Gallery. Select the gallery.

            Mosaic. Select between horizontal and vertical versions for mosaic view.

            Resizable mosaic. Enable the option to change the mosaic dimensions for the responsiveness purposes.

            Total width of mosaic. Set the mosaic gallery area percentage in relation to the overall content area.

            Images per page. Set the number of images to be displayed in a single page.

            Image title. Choose to display the image titles, display on hover or not to display it.
