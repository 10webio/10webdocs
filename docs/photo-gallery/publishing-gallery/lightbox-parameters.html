            Lightbox Parameters

            Publishing Photo Gallery. Lightbox Parameters

            Full width lightbox. Enable full width features for the lightbox.

            Lightbox dimensions. Define the lightbox height and width in pixels.

            Lightbox effect. Select the lightbox effect from the provided options.

            Lightbox autoplay. Choose whether to autoplay the lightbox or not.

            Time interval. Set the time interval between the display of the sequential image.

            Enable filmstrip in lightbox. Choose whether to have filmstrip enabled in the pop-up or not.

            Filmstrip height. Define the height of the filmstrip if applicable.

            Enable control buttons. Choose whether to have control buttons enabled in the lightbox or not.

            Enable fullscreen. Choose whether to have Fullscreen enabled for the lightbox or not.

            Enable comments. Choose whether to display comment box in the pop-up or not.

            Enable Facebook button. Choose whether to have Facebook social sharing button enabled in the pop-up or not.

            Enable Twitter button. Choose whether to have Twitter social sharing button enabled in the pop-up or not.

            Enable Google+ button. Choose whether to have Google+ social sharing button enabled in the pop-up or not.

            Enable Pintererst button. Choose whether to have Pinterest pin button enabled in the pop-up or not.

            Enable Tumblr button. Choose whether to have Tumblr social sharing button enabled in the pop-up or not.

            For Slideshow View only:

            Enable image title. Choose whether to display image titles during the slideshow or not.

            Title position. Set the position of the title using the available options.

            Enable image description. Choose whether to display image description during the slideshow or not.

            Description position. Set the position of the description using the available options.

            Enable slideshow music. Choose whether to have background audio track during the slideshow or not.

            Music URL. Provide the URL of the audio track to play during the slideshow.

            For Carousel View only:

            Carousel dimensions. Set up maximum Carousel dimensions.

            Carousel image ratio. Define the proportions between the main image and images preceding and following the main image in the carousel roll.

            Enable Autoplay. Choose whether to enable the autoplay of the carousel roll or not.

            Time interval. Set time for shifting images included in carousel roll if autoplay is enabled.

            Enable Image Title. Image title will be only visible on main image if activated.

            Fixed width. Define the maximum width of the carousel display. The width will depend on the content area, where carousel is inserted. The visible images of carousel view will automatically adapt to this width.

            Container fit. If you activate the option the distance between the images will be identical so that the images fill the container, otherwise the distance between the images will depend on the ratio.

            Next/Previous buttons. Choose whether to include navigation buttons for the roll or not.

            Play/Pause button. Choose whether to display play/pause buttons over the carousel roll or not.
