      Managing and Publishing Spider Contacts

      Instructions to configure, manage and publish Spider Contacts

      Creating categories

      On the left menu select Spider Contacts >Categories.

      From Spider Contact Manager **choose **Categories.

      In the upper right corner click on the Add a category button.

      Name. Add a name for the category.

      Description. Add a description for the category.

      Parameters. Here you can define different parameters for the category. You can add as many parameters as you wish. After filling in the current parameter title, press “Enter” to proceed to the next parameter.

      Order. Choose the order of the category.

      Published. Choose whether to publish the category or not.

      Creating Contacts

      On the left menu select Spider Contacts > Contacts.

      In the upper right corner click on the Add a Contact button.

      First Name. Add the first name of the contact.

      Last Name. Add the last name of the contact.

      Email. Add the Email.

      Send Email when Message Sent. Choose whether the submitted messages are redirected to the contact’s e-mail or not.

      Category. Choose a category from the existing list of categories.

      Images. You can add an unlimited number of images to the contact.

      Description. Add a description for the contact.

      Order. Choose the order of the contact.

      Published. Choose whether to publish the contact or not.

      Messages

      In this section you can view all the submitted messages.

      On the left menu select Spider Contacts > Message.

      Select a category for the message.

      Select a contact for the message.

      Each message in the list has the following parameters: Message title, Sender name, Phone, Sender, Contact Preferences with Sender, Category, and Date.

      It is possible to filter the messages by the date and name parameters.

      Publishing the Created WordPress Contacts

      To insert a contact into a Page or a Post.

      Press the button named Insert Spider Contacts in a post or a page you want to show the contact.

      Single Contact

      Select Contact. Select a contact from the given list.

      Contacts List Category

      Select Category. Select a category from the given list.You can either select “All Categories” or choose single/multiple categories from the list.

      Contacts display type. Choose whether to display the contacts in a short view, table view or Full view.

      Order by. Choose the ordering basis from the provided options (id, Ordering, Firstname, LastName).

      If there is any problem with editor, you can add the short code manually to the pages or posts.

      Single Contact

      Add the short code [Spider_contact_single id="my id"] to the pages or posts and change the following point.

      • ID. Add the ID (Spider Contacts>Contact >ID) of the single contact to the appropriate place (“my id”) in the shortcode.

      Contacts List Category

      Add the short code [Spider_contact_categories id="Contact ID" type=""order_by=""] to the pages or posts and change the following points.

      • ID. Add the ID (Spider Contacts>Contacts >ID) of the category which you want to show to the appropriate place (“my id”) in the shortcode. If the value of the ID is All_CAT, all categories will be shown. If you want to add multiple IDs, then you need to list them separating by commas.
      • Type. If the value of the type is 0, the contacts list category will be shown in short view. If the value of the type is 1, the contacts list category will be shown in a table view. If the value of the type is 2, the contacts list category will be shown in a Full view.
      • Order by. If the value of the order by is 1, the contact list will be ordered by id field. If the value of the order by is 2, the contact list will be ordered by the Ordering field. If the value of the order by is 3, the contact list will be ordered by the Firstname field. If the value of the order by is 4, the contact list will be ordered by the Lastname field.
