        Spider Video Player Themes

        Instructions to modify theme settings Spider Video Player plugin

        This section allows adding or editing the default themes for the WordPress Video plugin. Below you will find the detailed descriptions of the options in various parameter groups of the Theme.

        Creating A Theme

        On the left menu select Video Player > Themes

        In the upper left corner click on the Add a Theme button.

        General Parameters

        Title of theme. Choose a title for the theme.

        Width of player. Define the width of the player.

        Height of player. Define the height of the player.

        Start with. Define the mode of the player at start (Video or Playlist).

        Show Track Id. Choose whether to show the sequential number of the track in the playlist.

        Auto hide time. Define how long the control bar and the playlist stay on screen while being inactive.

        Keep aspect ratio. Choose whether to keep the original aspect ratio of the video or not.

        Control bar over video. Choose whether the control bar appears on the video area or above/below it.

        Watermark image. You can add an image to appear as a watermark on the screen.

        Watermark position. Choose a position for the watermark image.

        Watermark size. Select a size for the watermark image.

        Watermark margin. Select the distance between the watermark image and the edge of the player window.

        Watermark transparency. Select the transparency level for the watermark image.

        Style Parameters

        Center button transparency. Define the transparency level of the play/pause button at the center of the screen (‘0’ for complete transparency).

        Background color. Choose the main background color.

        Video background color. Choose the color of the video background.

        Frames background color. Choose the background color of the control bar and the playlist.

        Frames background transparency. Define the transparency level of the control bar and the playlist.

        Control buttons main color. Select the color of the control buttons.

        Control buttons hover color. Select the color of the hovered control buttons.

        Control buttons transparency. Select the transparency level of the control buttons.

        Sliders color. Select the color of the sliders.

        Hovered item background color. Choose what color a playlist item gets when being pointed on by the mouse cursor.

        Selected item background color. Choose what color an item gets after being selected.

        Text color. Select the default text color for the player.

        Hovered text color. Select the color for the hovered text.

        Selected text color. Choose the color of the selected text.

        Loading animation type. Choose what animation is being shown while loading the video.

        Playback Parameters

        Auto play. Choose whether the player automatically starts the playback on page opening or not.

        Auto next song. Choose whether it automatically switches to the next track in the playlist when the previous one is finished or not.

        Auto next album. Choose whether it automatically switches to the next playlists when the previous one is finished or not.

        Default volume. Define the default sound volume of the player.

        Repeat. Choose whether the player repeats the current track or the whole playlist by default or not.

        • Off: Do not repeat
        • One: Repeat the current track
        • All: Repeat the whole playlist

        Shuffle. Choose whether it shuffles the tracks by default or not.

        Control bar auto hide. Enable or disable control bar auto hide.

        Playlist and Library Parameters

        Playlist position. Choose whether the playlist appears on the left or on the right side of the player.

        Width of playlist. Define the width of the playlist.

        Playlist over video. Choose whether the Playlist appears on the video area or at the left/right side of it.

        Open Playlist at Start. Choose whether to automatically open the playlist after starting the player.

        Playlist auto hide. Enable or disable the option of automatically hiding the playlist.

        Playlist text size. Define the size of the playlist text.

        Library columns. Define the number of columns in the library.

        Library rows. Define the number of rows in the library.

        Library list text size. Define the size of the text in the library list.

        Library details text size. Define the size of the text in the library details.

        Video Control Parameters

        Play/Pause on click. Enable or disable play/pause by clicking on the player screen.

        Play/Pause by space key. Enable or disable play/pause by pressing the Space key.

        Volume control by mouse scroll. Enable or disable sound volume control through the mouse scroll.

        Control bar position. Choose whether the control bar is at the top or at the bottom of the player window.

        Buttons order on control bar. Select the order of buttons on the control bar by dragging and dropping them from one position to another. Click on a button to enable/disable it.
