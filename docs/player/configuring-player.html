        Configuring Spider Video Player

        Instructions to add tags, videos, playlists and create players with Spider Video Player plugin

        Adding Tags to WordPress Player

        In this section you’ll learn how to add tags to the WordPress Video plugin.

        The Tag is a custom parameter of videos by which you can search videos while creating playlists. Here are some examples of Tags: Year, Date, Artist, Album, Genre, Location, Country, Section etc.

        On the left menu select. Video Player > Tags

        In the upper left corner click on the Add Tag button.

        Name. Define a name for the tag (e.g. Artist).

        Required. Choose if the tag has to be defined for each video added or not.

        Published. Choose whether to publish the tag or not.

        Adding Videos

        On the left menu select. Video Player > Videos In the upper left corner click on the Add video button. Title. Add a title for the video. Type.

        • HTTP – Select the video (URL) and its HD version (UrlHD). You can provide the URL of the video or select it from the uploaded videos. It is also possible to add HTML5 videos (supports only MP4, WebM, and Ogg formats) using the Url(HTML5) and UrlHD(HTML5) options.
        • YouTube – Provide the youtube link to the video in the URL field.
        • RMTP – Fill in the Flash Media Server (FMS) URL of the video along with the regular URL and UrlHD (HD version).

        Thumb. Select an image for the video thumbnail out of the uploaded images. Use the “Remove Image” option to remove the selected image. Tags. Fill in the available tags for your video (see Step 2 for adding tags).

        Creating Playlists

        In this section you’ll learn how to create playlists in the WordPress Video Player.

        On the left menu select. Video Player > Playlist

        In the upper left corner click on the Add video button.

        Title. Choose a title for the playlist.

        Videos. Use the Add button to add videos to your playlist from the list of existing videos.

        Thumb. Select a thumbnail image for the playlist. Use the “Remove Image” option to remove the selected image.

        Adding a WordPress Player

        On the left menu select. Video Player > Spider Video Player.

        In the upper left corner click on the Add player button.

        Priority. Choose if you want to add a Flash or HTML5 player. Note that if the device (e.g. mobile phone) does not support Flash, the player will automatically switch to HTML5 mode (videos of unsupported formats will not be displayed).

        Title. Choose a title for the player.

        Select Theme. Select a theme from the existing list.

        Playlist. Use the Add button to add playlist to your player.
