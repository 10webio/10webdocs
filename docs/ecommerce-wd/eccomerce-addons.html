              Ecommerce WD Add-ons

              Description of Ecommerce WD Add-ons

              Ecommerce Filters

              Do you plan to showcase large number of products with Ecommerce WD? Don’t want your users to get lost? Want to make the product access easier and more user-friendly? Then Ecommerce Filters add-on is the solution you should be looking for.

              Attention: Ecommerce Filters Add-On requires Ecommerce WD Free version 1.0.13 and higher or Pro version 2.0.13 and higher: if you are using an older version, please update your plugin. To update the plugin please login at web-dorado.com, go to My Products and download the latest version.

              The add-on will help you to add filtering tools to widget area. Using the options you can filter the product list within its categories and subcategories. Plus it will not take too much space.

              Filtering tabs can be added both on Ecommerce page and other pages on your website.

              Filtering tools include all of the necessary look-up options:

              • Filter by product name
              • Filter by category
              • Filter by manufacturer
              • Filter by price
              • Filter by date added
              • Filter by minimum rating
              • Filter by tags

              Ecommerce Mini Cart

              Mini cart is a superb tool for helping out clients with choices and summing up the orders. All of the products added to the shopping cart will be displayed at a custom location using the widget. The user will get a chance to do the following actions - checkout the products, remove one or more items from the list and get redirected to the shopping cart.

              The widget will be displayed on all pages except shopping cart and checkout pages.

              Attention: Ecommerce Mini cart Add-On requires Ecommerce WD Free version 1.0.15 and higher or Pro version 2.0.15 and higher: if you are using an older version, please update your plugin. To update the plugin please login at web-dorado.com, go to My Products and download the latest version.

              The add-on packs the following features:

              • Product names and thumbnails, parameters
              • Quantity of the added items
              • Separate prices for the added products
              • Subtotals for each item type
              • Total price of the added items

              Ecommerce Latest Products

              Do you want to keep your clients updated with new arrivals? Then Ecommerce Latest Products add-on is just for your case. The add-on allows displaying latest products based on a single item (narrow reach) or multiple categories of your choice (wider reach). You can showcase the product list defining the number of available items. The user will get a chance to quick preview the item, checkout the product or get redirected to the main product page.\

              Attention: Ecommerce WD Latest Products Add-On requires Ecommerce WD Free version 1.1.4 and higher or Pro version 2.1.4 and higher: if you are using an older version, please update your plugin. To update the plugin please login at web-dorado.com, go to My Products and download the latest version.

              The add-on can display the following:

              • Product name, thumbnail, price
              • QuickView button
              • Comparison button
              • Product Rating
              • Buy Now/Add to Cart buttons

              Top Rated products

              Customer feedback is one of the most outstanding components for online stores. That’s why Ecommerce WD allows product rating. To highlight the importance of the rating you can use Top Rate products add-on. The add-on will allow displaying top rated items of single or multiple categories, depending on the setup. The number of displayed items can be customized. The widget will give access to the product main page, quick preview and checkout.

              Attention: Ecommerce WD Top Rated Products Add-On requires Ecommerce WD Free version 1.1.5 and higher or Pro version 2.1.5 and higher: if you are using an older version, please update your plugin. To update the plugin please login at web-dorado.com, go to My Products and download the latest version.

              Overall the add-on includes the following parameters:

              • Product Rating
              • Product name, thumbnail, price
              • QuickView button
              • Comparison button
              • Buy Now/Add to Cart buttons

              Ecommerce Bestsellers

              In most of the cases best sellers display leads to increase of store sales. Ecommerce Bestsellers add-on helps you out with this. It will allow displaying best-selling items and products in demand in a custom position. The items can be displayed based on single or multiple categories, depending on setup. The number of displayed items can be customized.

              Attention: Ecommerce WD Bestsellers Add-On requires Ecommerce WD Free version 1.1.6 and higher or Pro version 2.1.6 and higher: if you are using an older version, please update your plugin. To update the plugin please login at web-dorado.com, go to My Products and download the latest version.

              The add-on comes with the following parameters:

              • Product name, thumbnail, price
              • QuickView button
              • Comparison button
              • Buy Now
              • Add to Cart buttons

              Stripe

              Stripe add-on extends eCommerce WD payment options allowing you to take payments directly on your store using Stripe payment method. The method is widely used in US, Canada, UK and Australia. The payment can be received by users from 25 countries (Australia, Canada, Denmark, Finland, France, Ireland, Japan, Norway, Singapore, Spain, Sweden, United Kingdom, United States, Austria, Belgium, Germany, Hong Kong, Italy, Luxembourg, Netherlands, New Zealand, Portugal, Switzerland), but payments can be sent from any point. Direct payments using Credit Cards are accepted by Stripe.

              It is required to have an SSL certificate in order to use Stripe, since the transaction is processed within the website. Though provided card data is not being stored within the website (it gets stored in Stripe servers) it is necessary to have a secure website.

              Attention: Ecommerce Stripe Add-On requires Ecommerce WD Free version 1.2.2 and higher or Pro version 2.2.2 and higher: if you are using an older version, please update your plugin. To update the plugin please login at web-dorado.com, go to My Products and download the latest version.

              Stripe Payment System Set up

              To access the options go to Ecommerce WD > Options > Checkout. Then press Payment Systems button and select Stripe. Fill in the details below and press Save button.

              Checkout mode. Choose to have Stripe set in test or live (production) mode. Fill in Live or Test API keys based on the checkout mode selected. API keys can be found following the instructions.

              Saved Cards. If enabled user Credit Card data is getting attached to the WordPress user, thus when logged in, the list of filled in Credit Cards (last 4 digits) will be displayed to the user. Plus the user will get a chance to use an optional credit card adding it to the list.

              Stripe Checkout. Instead of the default inline payment form, a pop-up is used to collect credit card details. You can have a custom image for the checkout pop-up icon.

              Capture Charge Immediately. The payment for the item can be captured at one of the following steps: when the item is purchased or when the payment is made and confirmed by the administrator. If enabled, you opt for the first option. Note that in this case administrator will confirm the payment with a Capture button located at Orders > Payment Status.

              Published. The payment method can be unpublished here to use one of existing options.
