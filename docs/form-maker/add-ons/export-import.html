                  Form Maker Export/Import Extension

                  Moving your forms and submissions to a different location doesn’t have to be complicated and frustrating. With Import/Export extension of WordPress Form Maker plugin, you can export your forms and import them to another website with just a few simple steps. Extend the features of Form Maker plugin – get Import/Export extension today!

                  Attention: Import/Export extension works with Form Maker version 1.7.84/2.7.84 or higher. If you are using an older version, please update your plugin.

                  Export/Import is a capacity-enhancing extension designed for WordPress Form Maker plugin. It comes packed with three main features:

                  • exporting forms,
                  • exporting forms and submissions,
                  • importing the exported forms.

                  The product is easy-to-use and does not require any coding experience. To begin using the options mentioned above, install the extension to your website, then navigate to Form Maker > Forms page.

                  As the extension is installed and activated on your WordPress site, three buttons will be added at the top of Form Maker > Forms page:

                  • Export,
                  • Export with submissions,
                  • Import.

                  Let’s discuss each feature to discover the benefits and possibilities.

                  Exporting Forms

                  Begin transferring WordPress forms from one website to another by exporting them. This can be done with just a few clicks. Select the forms you wish to download, then press Export button. An XML file will start downloading instantly, containing the structure and data of the selected forms.

                  Note: Depending on the size of your form, the download can take from a couple of seconds to a few minutes.

                  Exporting Forms with Submissions

                  Export/Import extension also lets you download the data of your forms along with the submissions users made. To export your forms with this method, mark the items you wish to download, then click Export with submissions button.

                  The downloaded XML file will contain the selected forms and corresponding submissions.

                  Importing the Exported Forms

                  To import the downloaded forms into another WordPress website, make sure to have both Form Maker and Export/Import extension installed. Navigate to Form Maker > Forms page and press Import button. A popup box will appear, allowing you to browse the exported XML file. As you find the file with Choose File button, press Import.

                  Note: Depending on the size of your forms, the process of importing can take up to a few minutes.
