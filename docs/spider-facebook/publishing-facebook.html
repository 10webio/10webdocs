        Publishing Spider Facebook in a Page or a Post

        Instructions to publish the created Facebook into a page or post

        Open the page/post where you want to publish the Facebook plugin. Click on the Spider Facebook button.

        Select the Facebook plugin you want to insert and click on the Insert button. You can also edit the META tags in the Spider Facebook section at the bottom of the page. Note that these META tags have a higher priority than the ones on the Facebook plugin parameters page.

        Alternatively, you can manually insert the Facebook pluginshortcode into the post/page. Simply copy [spider_facebook id="N"] into the page/post. “N” is the ID of the plugin and can be found in the list of plugins in the Manage Facebook section of Spider Facebook.
