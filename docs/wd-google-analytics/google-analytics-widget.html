                  Publishing Google Analytics Widget

                  WordPress Google Analytics plugin lets you place a statistics widget on the front-end of your website. You can display the website sessions, as well as current page traffic to users visiting your website.

                  To begin configuring the widget, navigate to Appearance > Widgets page. Look for Google Analytics WD, then add to the widget area where you wish to have it.

                  Set a Title for the widget as the first step. Afterwards, select the Default Date for displaying corresponding statistics. You can choose one of the following options:

                  • All Time,
                  • Last 30 Days,
                  • Last 7 Days,
                  • Last Week,
                  • This Month,
                  • Last Month,
                  • Yesterday,
                  • Today.

                  Furthermore, you can choose the data to display with Google Analytics widget. Use Widget report data drop-down box to select between the following options:

                  • Site data,
                  • Current URL data.

                  As you finish configuring these settings, make sure to press Save.

                  Note: Google Analytics widget displays statistics from the current WordPress website only.
